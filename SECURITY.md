# Security Policy

Agentic Command is pre-1.0 software. Security fixes are provided for the latest
release only.

Please report suspected vulnerabilities privately through GitHub's **Report a
vulnerability** security-advisory flow for `jvdub/agent-command`. Do not include
terminal transcripts, credentials, API keys, or private workspace files unless
they are strictly necessary and have been redacted.

For ordinary defects, use the public issue tracker.
